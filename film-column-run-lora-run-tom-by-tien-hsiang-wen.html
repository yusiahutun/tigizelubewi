光陰似箭，橫空出世的德國電影《蘿拉快跑》（Run, Lola, Run）已經是20年前的事了。往新格局馳騁

這部電影的情節其實很簡單：蘿拉接到男友電話，說他弄掉要交給老大的十萬贓款，眼看碰面時間就要到了，蘿拉只剩20分鐘籌錢解救她的男友；然後我們就看她開始不停地跑、跑、跑……。

第一個20分鐘過去了，她無力挽回男友搶劫超市的笨舉，只好一起加入，自己卻死於警察槍下。

「不對！」倒在血泊中的蘿拉不甘願如此結局。

OK，電影回到原點，重新再來：蘿拉接到男友的電話……只剩20分鐘……開始跑……。遇到的人物、經過的風景都一樣，卻出現了三個不同的結局，直到happy ending為止！

此內容來源為 youTube，該網站能找到相關內容或更多資訊。

人生絕少可以重來，電影卻只消一些底片或更大容量，就可以重新來過。

與其說導演Tom Tykwer非把它拍到喜劇收場才罷手，不如說是藉由三個同樣處境卻不同收尾的際遇，來闡述世事的變化莫測，一個小小的決定或時間差，就可能影響結果，甚至人的一生。

你是會震驚生命宇宙的因果關連，還是把死亡重來當成電子遊戲看待呢？

我第一次看《蘿拉快跑》的時候，想到的是波蘭大師Krzysztof Kieslowski在1981年拍攝（但1987年才得以上映）的《機遇之歌》（Blind Chance），後者描述一個醫學院學生在月台追趕已經啟動的火車，也出現三個截然不同的結果。但Tom Tykwer凌厲的節奏與去道德化的角色，風格上反而更接近那時火紅的《猜火車》的Danny Boyle以及甫完成《第五元素》的Luc Besson。

Rex

《機遇之歌》，1987。

有趣的是後來Tom Tykwer卻成為第一個把Krzysztof Kieslowski（大師在1996年過世）遺留人間的劇本《天堂奔馳》拍成電影的人，並由Cate Blanchett主演。看來這麼聯想的人不只是我。這也順勢開啟了他跨國、英語電影的拍攝。

Rex

《天堂奔馳》，2002。

回歸個性本質

所以無需意外暢銷小說改編的《香水》才是Tom Tykwer在台灣最賣座的作品。但論創意與影響力，還是難以和《蘿拉快跑》相提並論。

Rex

《香水》，2006。

當時用滿頭紅髮點燃銀幕的女主角Franka Potente奮力激昂的姿態，讓人對她的愛情意志動容，亦為那個世代的叛逆虛無寫下適切的註腳。之後Franka Potente也轉往好萊塢和Matt Damon合演了《神鬼認證》系列前兩集。

Rex

《雲圖》，2012。

Tom Tykwer近年和《駭客任務》的華卓斯基姐弟（The Wachowskis）聯合編導了命運多舛的《雲圖》; 這部預算超過一億美金的電影評價兩極，在中國上映被刪減將近40分鐘也引來譁然。不過男主角Tom Hanks應該跟他合作很愉快，兩人後來又拍了《梭哈人生》。

Getty Images

擔任2018年評審團主席的導演Tom Tykwer，於柏林影展中留影。

Tom Tykwer也是今年柏林影展評審團主席，他與媒體意見相左地拔擢了羅馬尼亞女導演Adina Pintilie混淆劇情、紀錄、實驗去探索身體感官的長片處女作《禁身接觸》（Touch Me Not），卻讓德國電影全軍覆沒。

過了20年，即使作品已不復當年犀利，Tom Tykwer帶領風潮或引發爭議的本領還是不小啊！

聞天祥

16歲開始發表電影文字，20歲起為臺灣各大報撰寫影評及專欄，投入影評書寫已30年。文字之外，曾擔任台北電影節策展人，也受邀出任國內外電影獎評審，現為台北金馬影展執行委員會執行長。
